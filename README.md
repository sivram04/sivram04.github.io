Personal Portfolio Website

This repository contains the source code for my personal portfolio website, built to showcase my background in Data Analytics, Business Analysis, and Data Engineering.
The website highlights my skills, projects, experience, certifications, and provides a way to contact me directly.

🔗 Live Website:
👉 https://sivram04.github.io
Features

Clean, professional, recruiter-friendly design

Responsive layout for desktop and mobile

Sections for:

About

Skills

Projects

Experience

Education

Certifications

Extra-curricular activities

Contact form with email notifications (Formspree)

Smooth scroll and subtle reveal animations

Hosted using GitHub Pages

Tech Stack

HTML5 – Structure and content

CSS3 – Styling, layout, animations

JavaScript (Vanilla) – Interactivity and scroll effects

Git & GitHub – Version control

GitHub Pages – Deployment

Formspree – Contact form handling (no backend required)

Deployment

This website is deployed using GitHub Pages.

Steps used for deployment:

Push static files (index.html, css/, assets/) to GitHub

Enable GitHub Pages from the main branch

Site becomes available at:
https://sivram04.github.io
